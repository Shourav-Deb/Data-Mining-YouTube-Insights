# Data Mining Project (YouTube Insights)

A compact showcase of a full YouTube text mining workflow built through step by step single different tasks and then extended into a final project structure.

## Overview

This repository presents a connected data mining pipeline using YouTube captions and comments. The work starts from raw data collection, moves through text cleaning and preprocessing, and then continues into pattern discovery, temporal analysis, clustering, and final synthesis between captions and audience comments.

The project is organized as a practical learning repository: it includes completed lab-based work, project guidance, extra supporting materials, and validation questions for practice and self-checking.

## What This Repository Contains

- **Project instruction manual:** covering the full end-to-end workflow and final project direction.
- **Task-by-task lab work:** organized around the main stages of the project.
- **Implemented code files:** for the core data mining tasks completed across the labs.
- **Reports / outputs for each task:** to help readers understand the process and results.
- **Output plots:** to help understand the outcome plots.
- **Validation materials:** for practicing concepts and checking understanding based on the completed work.

## Project Workflow

The repository follows a step-by-step pipeline:

1. **Collection & Exploration** – gather YouTube captions and comments.
2. **Cleaning & Preprocessing** – normalize, tokenize, filter, and prepare reusable cleaned data.
3. **Pattern Discovery** – explore frequent words, co-occurrences, itemsets, and association rules.
4. **Temporal Analysis** – track how discussion patterns change across chunks of data.
5. **Clustering** – group comments into thematic clusters using text vectorization.
6. **Synthesis & Comparison** – compare captions and comments through keywords, n-grams, sentiment, and linked analysis.
7. **Final Project Assembly** – scale the workflow into a larger comparative project with structured reporting.

## Main Topics Covered

- YouTube text data collection
- Text cleaning and preprocessing
- Exploratory data analysis
- Frequent pattern and co-occurrence mining
- Incremental / trend analysis
- TF-IDF vectorization
- K-Means and DBSCAN clustering
- Keyword, bigram, and sentiment comparison
- Final project implementing and validation practice

## Purpose of This Repository

This repository is mainly a **showcase + learning resource**. It is intended to help others see how a complete YouTube-based data mining project can be built step by step. It can also be used for practice, revision, and reference while working on similar text mining tasks.
